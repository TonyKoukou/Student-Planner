# Student Planner

## Περιγραφή
Python πρόγραμμα για διαχείριση μαθημάτων και εργασιών. 
Ο χρήστης μπορεί να προσθέσει μαθήματα, να προσθέσει εργασίες σε αυτά και να δει όλες τις εργασίες. 
Τα δεδομένα αποθηκεύονται σε αρχείο `planner.txt`.


## Οδηγίες Χρήσης
1. Βεβαιώσου ότι έχεις εγκατεστημένη Python 3.
2. Τρέξε το πρόγραμμα:cmd
3. Γράψε cd και την τοποθεσία του αρχείου π.χ cd C:\Users\Tony\Student Planner
4. Γράψε την εντολή python student_planner.py


## Μολίς ανοίξει θα σας εμφανίσει: 
1 για προσθήκη μαθήματος
2 για προσθήκη εργασίας
3 για προβολή μαθημάτων
4 για έξοδο

## Όλα τα δεδομένα αποθηκεύονται στο αρχείο planner.txt.
